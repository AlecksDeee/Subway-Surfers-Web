# Local Set-up
1. Clone the repository `git clone https://github.com/AlecksDeee/Subway-Surfers-Web.git`
2. Open up the `host.py` file
3. Follow the localhost link
4. Done!

# Codespaces Set-up
1. Make a codespace workspace
2. In terminal write `python host.py`
3. Follow the localhost link
4. Done!
